## <font color="#F2853F" style="font-size:24pt"> os_sched_wakeup </font>

```c
int
os_sched_wakeup(struct os_task *t)
```

Called to make task *ready to run*. If task is *sleeping*, it is woken up.


#### Arguments

| Arguments | Description |
|-----------|-------------|
| `t` | Pointer to task |

#### Returned values

Returns 0 on success.
