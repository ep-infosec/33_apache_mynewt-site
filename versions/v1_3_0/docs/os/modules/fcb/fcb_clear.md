## <font color="F2853F" style="font-size:24pt">fcb_clear</font>

```no-highlight
int fcb_clear(struct fcb *fcb);
```

Wipes out all data in FCB.

#### Arguments

| Arguments | Description |
|-----------|-------------|
| fcb | Points to FCB. |


#### Returned values

Returns 0 on success; non-zero otherwise.
